# vuejs3-linkedin-project

## Configuration du projet
```
npm install
```

### Compiler et recharger pour le développement
```
npm run serve
```

### Compiler et réduire pour la production
```
npm run build
```

### Lints et corriger les fichiers
```
npm run lint
```
